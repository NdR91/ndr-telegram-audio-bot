Provider ≠ Modello — cosa cambia
  • Un provider è una connessione API (OpenAI, Gemini, OpenRouter…).
  • Un modello è un'AI specifica offerta da un provider (es. gpt-4o, whisper-1).
  • Stesso provider, modelli diversi = capacità diverse.
  • Esempio: con OpenRouter, whisper-1 per trascrizione e gpt-4o per refinement.
  • I fallback permettono di specificare modelli alternativi in caso di errore del primario.
